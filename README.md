# Pomos: Focus Timer with Stats

A simple, flexible pomodoro timer web app, with an optional feature to track the time you spent focusing. Built with React and Supabase.

## Features

- Customizable work/break lengths
- Interactive charts of your timer usage
- Light/dark mode toggle
